# E-Commerce-Orders-2026
E-commerce order analysis and sales prediction using Random Forest Classifier and Regressor with Python and Scikit-learn.
# E-commerce Orders Dataset 2026

## Overview
This project explores e-commerce order records and builds 
machine learning models to predict order value and classify 
high-value orders using Python in Google Colab.

## Dataset
- E-commerce orders data 2026
- Target 1: High_Value_Order (0 = No, 1 = Yes)
- Target 2: Order_Amount (regression)

## Key Columns
| Column | Description |
|--------|-------------|
| Product_Category | Type of product ordered |
| Order_Amount | Total order value |
| Traffic_Source | How customer reached the site |
| Membership_Status | Customer membership level |
| High_Value_Order | Target column |

## Steps
1. Data Loading and Exploration
2. Handling Missing Values
3. Preprocessing (OneHotEncoding)
4. Visualization (bar plots, distributions)
5. Random Forest Classifier
6. Random Forest Regressor
7. Evaluation (Accuracy, MAE, R² Score)

## Technologies
- Python, Pandas, NumPy
- Matplotlib, Scikit-learn
- Google Colab

## Results
- Classifier: Random Forest (High Value Order Prediction)
- Regressor: Random Forest (Order Amount Prediction)
- R² Score: ~0.90
